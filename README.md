## About me

I'm **Vexoffx**, a self-taught Moroccan developer based in Italy. I turn curiosity into real projects across gaming, tooling, automation, and clean product interfaces.

- 🎮 Building Minecraft clients, mods, launchers, and game tools
- 🌐 Creating fast portfolio sites and polished web experiences
- ⚙️ Exploring automation and AI-assisted workflows
- 🧠 Focused on clean structure, speed, and useful products

<br clear="right"/>

## My story

My journey started in **2023**, when a gaming community made me curious about how software works. I built foundations in **C#**, moved into lower-level thinking with **C++**, created my first Minecraft client in **2025**, and now use Java, JavaScript, Python, and AI-assisted workflows to ship stronger prototypes.
