## About me

I'm **Vexoffx**, a self-taught Moroccan developer based in Italy. I turn curiosity into working products across gaming, tooling, automation, and clean interfaces.

- 🎮 Minecraft clients, mods, launchers, and game tools
- 🌐 Fast portfolio sites and polished web experiences
- ⚙️ Automation prototypes and AI-assisted workflows
- 🧠 Clean structure, speed, and useful products

> **Respect the user.** Tools should feel fast, predictable, and clear. Good design is part of good engineering.
